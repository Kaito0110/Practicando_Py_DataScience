# Practicando_Py_DataScience
# Alura Store Challenge

## Descripción del proyecto

Este proyecto realiza un análisis de datos de ventas de las cuatro tiendas de Alura Store con el objetivo de identificar cuál de ellas presenta el rendimiento más bajo y, con base en eso, recomendar cuál tienda sería la mejor opción para vender e invertir en un nuevo negocio.

El análisis se hizo en Python usando Google Colab y pandas, tomando en cuenta métricas clave de desempeño de cada tienda.

---

## Propósito del análisis

El propósito de este análisis es comparar el rendimiento de las cuatro tiendas a partir de distintos indicadores de negocio para apoyar la toma de decisiones.

Se analizaron los siguientes puntos:

- Facturación total por tienda
- Ventas por categoría de producto
- Calificación promedio de los clientes
- Productos más vendidos y menos vendidos
- Costo promedio de envío por tienda

Con estos datos se buscó detectar qué tienda tiene el desempeño más débil en comparación con las demás.
